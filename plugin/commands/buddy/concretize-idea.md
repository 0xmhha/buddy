# /buddy:concretize-idea

이 command는 `concretize-idea` skill을 즉시 invoke한다.

- Skill: `plugin/skills/concretize-idea/SKILL.md`

## 실행 지시

`Skill` 도구로 `concretize-idea` skill을 호출하라. 사용자가 전달한 인자는 그대로 skill에 입력으로 넘긴다.

```
$ARGUMENTS
```

추가 컨텍스트가 필요하면 skill 본문이 요구하는 입력 항목을 사용자에게 물어본 뒤 진행한다.
